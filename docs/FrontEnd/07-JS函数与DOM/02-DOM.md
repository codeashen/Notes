# DOM基本概念

DOM 是 JS 操控 HTML 和 CSS 的桥梁。

DOM（Document Object Model，文档对象模型）是 JavaScript 操作 HTML 文档的接口，使文档操作变得非常优雅、简便。

DOM 最大的特点就是将文档表示为节点树。

### 读后感

空出的时间看完本书，书中的代码也都敲了一遍，作者利用漫画这种方式增加读书的趣味性，内容浅显易懂，用于了解Python语法还可以吧，只是感觉很多地方用漫画有点牵强附会。

值得一提的是起初完全空读，很快就读了好几章节，过两天语法就忘了，所以之后把前面的代码都补上了，后面的章节也都是边度边Coding。这类书籍，不能完全脱离实践，急于进度。
